Used to get the most recent post from a user.

With a filter applied, this will return the most recent post (singular) that match the filter.

```curl noInline
GET https://api.itsjusttriz.com/bsky/latest/:handle
```

## Request Parameters

| Identifier |                     Description                     | Required |
| ---------- | :-------------------------------------------------: | -------: |
| :handle    | The `@` handle associated with the bluesky account. |       âœ… |

## Query Parameters

| Identifier |                                       Description                                       | Required |
| :--------- | :-------------------------------------------------------------------------------------: | -------: |
| filter     | Filters through the posts for certain content <br/>E.g. `?filter=twitch.tv/itsjusttriz` |      :x: |
| raw        |         Forces a JSON response, holding all related data.<br/>E.g. `?raw=true`          |      :x: |

## Example Responses
